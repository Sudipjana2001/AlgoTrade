# Indian Market Insights

## Project Info

This project provides insights into the Indian Market. It is a web application built with modern web technologies.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How to Run Locally

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/sudipjana525-bit/indian-market-insights.git
    ```

2.  **Navigate to the project directory:**
    ```sh
    cd indian-market-insights
    ```

3.  **Install dependencies:**
    ```sh
    npm install
    ```

4.  **Start the development server:**
    ```sh
    npm run dev
    ```

5.  **Build for production:**
    ```sh
    npm run build
    ```
